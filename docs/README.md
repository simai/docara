# Документация Docara

Пользовательская и техническая документация хранится как переносимый сайт
Docara в каталоге `docs/site`. Это единственный источник содержимого сайта;
`build_*` является генерируемым результатом и не должен редактироваться.

## Сборка

Из корня репозитория:

```bash
cd docs/site
php ../../docara build production
```

Результат появится в `docs/site/build_production`.

## Структура

- `site/docara.json` — настройки сайта и ссылка на точный Framework lock.
- `site/content` — русскоязычные Markdown-страницы и JSON sidecars.
- `site/simai-framework.lock.json` — зафиксированный runtime текущего
  проверяемого контура.

Детальные страницы скрыты из короткого глобального меню и открываются из
восьми страниц-разделов. При изменении кода Docara сначала обновляйте связанный
справочный материал, затем проверяйте исходные ссылки и собирайте сайт.
