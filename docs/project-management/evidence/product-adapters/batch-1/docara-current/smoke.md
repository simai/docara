# Smoke

Smoke scope:

- Composer metadata validates.
- PSR-4 autoload is regenerated.
- Docara contract classes autoload.
- Unit contract files execute through package-local test command.
- Scope checker confirms changes stay inside launch record allowed files and
  evidence path.

No public route, renderer, admin editor, search indexer or import/export smoke
exists in this batch because no runtime Docara surface is implemented.
