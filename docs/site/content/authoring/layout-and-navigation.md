# Макеты и навигация

Portable Docara использует два preset:

| Preset | Назначение |
| --- | --- |
| `docs` | Шапка, меню и область статьи |
| `landing` | Сфокусированная страница без меню документации |

`layout.max_width` принимает `compact`, `normal`, `wide` или `full`.
`settings.theme` принимает `system`, `light` или `dark`.

Для меню доступны два параметра:

- `navigation.hidden` исключает страницу из глобального меню;
- `navigation.order` задаёт неотрицательный порядок; при равенстве используется URL;
- страница без `navigation.order` располагается после страниц с явно заданным порядком.

Заголовок меню берётся из page sidecar, а если его нет — из первого Markdown
H1. Глубокие справочные страницы удобно скрывать на уровне `_section.json` и
связывать со страницей-разделом обычными ссылками.
