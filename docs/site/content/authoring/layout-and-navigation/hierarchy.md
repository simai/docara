# Иерархия разделов

Docara строит дерево навигации из структуры Markdown-файлов. Страница раздела
и одноимённая папка объединяются в один пункт: ссылка открывает обзор, а
отдельная кнопка раскрывает вложенные материалы.

## Глубина

Внутренняя модель не ограничивает количество уровней. Продуктовый интерфейс и
Simai Framework проверены на четырёх видимых уровнях; более глубокие уровни
сохраняют семантику и используют оформление четвёртого уровня.
