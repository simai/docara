# Markdown

Portable Docara поддерживает обычные заголовки, абзацы, списки, ссылки,
цитаты, fenced code, таблицы, зачёркивание и типографические преобразования.

```markdown
# Установка

1. Установите пакет.
2. Инициализируйте сайт.

| Команда | Назначение |
| --- | --- |
| `docara build local` | Локальная сборка |
```

Raw HTML удаляется, а небезопасные ссылки запрещаются. Для богатых элементов
используйте [подтверждённые компоненты](/components/), а не произвольную
HTML-разметку.

Директивы внутри fenced code остаются текстом, поэтому их можно безопасно
показывать в примерах.
