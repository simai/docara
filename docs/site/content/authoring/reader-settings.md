# Настройки чтения

Кнопка с иконкой настройки в шапке открывает компактную панель оформления.
Читатель выбирает один из трёх режимов:

- **Как в системе** — тема следует светлому или тёмному режиму устройства и
  меняется вместе с ним;
- **Светлая** — всегда используется светлая тема Simai Framework;
- **Тёмная** — всегда используется тёмная тема Simai Framework.

Выбор применяется сразу, сохраняется только в текущем браузере и действует при
переходах между страницами. Кнопка сохранения не нужна.

## Настройка сайта по умолчанию

Автор задаёт первый режим через наследуемую ветку `settings`:

```json
{
  "settings": {
    "theme": "system"
  }
}
```

Допустимы `system`, `light` и `dark`. Значение из `docara.json` можно
переопределить в `_section.json` или page sidecar; неизвестное значение
останавливает сборку.

Сохранённый выбор читателя имеет приоритет над настройкой страницы. Команда
«Вернуть настройку сайта» удаляет локальный выбор и сразу применяет текущее
унаследованное значение. Пока читатель ничего не переопределил, эта команда не
показывается.

## Хранение и приватность

Docara хранит только значение `system`, `light` или `dark` под ключом
`docara.reader.theme.v1` в локальном хранилище браузера. Аккаунт, сервер,
аналитика и внешний сервис не используются. Старое значение темы из cookie
`sf-theme` распознаётся для совместимости и очищается после нового выбора или
сброса.

Тема восстанавливается до первого отображения страницы, поэтому при переходе
нет промежуточной вспышки другой темы. Если браузер запрещает локальное
хранилище, страница продолжает работать с настройкой автора.

## Почему здесь нет других переключателей

Масштаб текста надёжно меняется средствами браузера и операционной системы.
Ширину страницы задаёт автор через `layout.max_width`, потому что она зависит
от таблиц, примеров кода и макета лендинга. Docara не создаёт второй
пользовательский layout-контракт без доказанной задачи.

См. также [макеты и навигацию](/authoring/layout-and-navigation/) и
[наследование настроек](/authoring/inheritance/).
