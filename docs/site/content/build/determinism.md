# Воспроизводимость

При одинаковых Markdown, JSON, Framework lock и версии генератора повторная
сборка должна создавать байт-в-байт одинаковый результат.

Docara не записывает в portable output текущую дату и абсолютные локальные
пути. Для каждой страницы `.docara/resolved-page-plans.json` содержит:

- итоговую конфигурацию;
- упорядоченный trace входных файлов с SHA-256;
- provenance каждого значения;
- canonical hash плана;
- нормализованные вызовы компонентов;
- точный asset plan;
- output path и публичный URL.

Для проверки соберите сайт дважды в чистые каталоги и сравните хэши деревьев.
Изменение входа или lock должно объяснимо изменить соответствующий план.

Если поиск включён, `content_sha256` в `_docara/search-index.json` вычисляется
из canonical documents, а query revision `_docara/search.js` — из точных байтов
runtime. Поэтому изменение содержания и изменение поискового кода инвалидируют
кэш независимо и остаются воспроизводимыми.
