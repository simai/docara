# Card

Card группирует связанное содержание. Это Markdown-рецепт с семантическим
`section`, границей, поверхностью и отступами Simai Framework.

:::card
### Переносимый проект

Markdown и JSON можно хранить рядом с кодом, проверять через Git и собирать
одной PHP-командой.
:::

## Код примера

```markdown
:::card
### Переносимый проект

Markdown и JSON можно хранить рядом с кодом.
:::
```

Внутри допустим обычный Markdown. Блок не принимает JSON props и не может
содержать другой `card` или `steps`.
