# CTA-ссылка

`:::cta` превращает одну Markdown-ссылку в основное действие со стилями
кнопки Simai Framework. Результат остаётся нативным `<a>`, поэтому переход
работает с клавиатуры и без JavaScript.

:::cta
[Начать работу](/start/)
:::

```markdown
:::cta
[Начать работу](/start/)
:::
```

Блок принимает ровно одну безопасную Markdown-ссылку с текстовой меткой.
Внутри метки разрешены text, emphasis, strong, strikethrough и перенос строки.
Обычный текст вне ссылки, image, inline code, HTML, несколько ссылок, вложенные
блоки и протоколы `javascript:`, `vbscript:`, `file:` или `data:` отклоняются
при сборке.

Smart-компонент `ui.button` только отображает action control: portable Docara
не выполняет действие, а у точного manifest нет параметра `href`. Для
навигации используйте `:::cta`.
