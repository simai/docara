# Список возможностей

`:::features` показывает от двух до шести пунктов как адаптивную сетку карточек.
До breakpoint `lg` Simai Framework используется одна колонка, начиная с
960 px — три.

:::features
- **Markdown.** Содержание остаётся читаемым исходным текстом.
- **JSON.** Настройки наследуются и проверяются схемами.
- **PHP.** Переносимая production-сборка не требует Node.js.
:::

```markdown
:::features
- **Markdown.** Содержание остаётся читаемым исходным текстом.
- **JSON.** Настройки наследуются и проверяются схемами.
- **PHP.** Переносимая production-сборка не требует Node.js.
:::
```

Тело должно содержать один плоский unordered list. Каждый пункт — один обычный
Markdown-абзац; в нём разрешены текст, ссылка, emphasis, strong,
strikethrough, inline code и перенос строки. Изображение, HTML, заголовок,
fenced code, несколько абзацев, вложенный список, один пункт или больше шести
пунктов отклоняются при сборке.
