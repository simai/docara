# Steps

Steps показывает короткую последовательность действий. Блок обязан содержать
один нумерованный Markdown-список.

:::steps
1. Подготовьте Markdown.
2. Проверьте JSON.
3. Соберите статический сайт.
:::

## Код примера

```markdown
:::steps
1. Подготовьте Markdown.
2. Проверьте JSON.
3. Соберите статический сайт.
:::
```

Обычный текст, маркированный список, пустой блок или вложенная Docara-директива
завершат сборку ошибкой.
