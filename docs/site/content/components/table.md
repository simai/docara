# Table

Таблица создаётся стандартным Markdown и получает Framework-классы таблицы,
границы и чередующихся строк. На узком экране контейнер прокручивается по
горизонтали, не расширяя всю страницу.

| Файл | Роль |
| --- | --- |
| `docara.json` | Настройки сайта |
| `_section.json` | Настройки раздела |
| `page.page.json` | Настройки страницы |

## Код примера

```markdown
| Файл | Роль |
| --- | --- |
| `docara.json` | Настройки сайта |
```

Это нативная таблица Markdown, а не `sf-table`: текущий Smart table требует
отдельного принятого data/settings contract.
