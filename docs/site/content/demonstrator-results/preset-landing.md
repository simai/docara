# Сфокусированная посадочная страница

Тот же источник Markdown можно показать без документационной навигации.

:::features
- **Короткое сообщение.** В центре остаётся цель страницы.
- **Один сценарий.** Читатель сразу видит следующий шаг.
- **Общий runtime.** Сборка использует тот же основной генератор.
:::
