# Product Smart-компоненты Docara

В шапке работает `docara.brand`, в левой области — `docara.navigation`, в
правой области — `docara.toc`. Секции макета только связывают эти компоненты с
данными страницы.

## Отдельный контракт

Каждый компонент имеет manifest, views, presets, шаблоны, Atlas, readiness и
собственные assets.

### Компонент владеет поведением

Навигация раскрывает активную ветвь и поддерживает до четырёх уровней. TOC
публикует событие перехода по заголовку. Общая оболочка не реализует это
поведение за компоненты.

## Единый реестр

`ui.*` и `docara.*` проходят общий manifest contract. Framework-компоненты
дополнительно проверяются по exact lock, а product-компоненты поставляются
через `DocaraSmartContribution`.

## Portable runtime

Готовый сайт остаётся статическим, а сборка не требует Laravel или Node.js.
