# Aside с контекстом страницы

Правая область получает заголовки из Markdown и дополняется уведомлением,
которое задаётся в JSON-композиции.

## Первый раздел

Заголовок становится ссылкой в автоматическом оглавлении.

## Второй раздел

Smart-компонент остаётся рядом с оглавлением и следует теме сайта.
