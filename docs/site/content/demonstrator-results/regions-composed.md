# Состав областей

Эта страница явно подключает фирменный header, многоуровневое меню слева и
оглавление справа. Основной контент остаётся обязательной областью макета.

## Основная область

Markdown попадает в `main`, а остальные области собираются из зарегистрированных
секций и Smart-компонентов оболочки.

## Правая область
