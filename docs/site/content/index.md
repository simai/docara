# Docara

Docara собирает статическую документацию, справочники и небольшие
содержательные сайты. Содержание остаётся в Markdown, настройки — в строгом
JSON, интерфейс строится на Simai Framework.

## Как устроен результат

:::card
### Файлы принадлежат автору

Markdown хранит содержание. `docara.json`, `_section.json` и page sidecar
описывают сайт, раздел и страницу. Производный `build_*` не редактируется.

:::

:::card
### Сборка объяснима

`.docara/resolved-page-plans.json` показывает итоговые настройки и источник
каждого значения. `_docara/component-catalog.json` фиксирует фактический набор
компонентов конкретной сборки.

:::

## Два вида страниц

Preset `docs` даёт дерево, breadcrumbs, outline и previous/next. Preset
`landing` использует тот же Markdown и JSON, но оставляет сфокусированный
макет без документационной навигации.

[Сравнить и настроить оба preset](/authoring/layout-and-navigation/).

## Границы документации

Описанный контур проверяется локальной сборкой. Документация не заявляет
production readiness, готовность всех компонентов Simai Framework, полностью
offline runtime или готовность публичного release.
