# Docara

Docara собирает статическую документацию и небольшие содержательные сайты.
Текст хранится в Markdown, настройки сайта и разделов — в строгих JSON-файлах,
а внешний вид строится на Simai Framework.

:::ui.alert
{"type":"info","variant":"default","title":"Один переносимый формат","supporting-text":"Авторам достаточно PHP, Markdown и JSON. Node.js нужен только разработчикам темы.","closable":false,"aria-label":"Один переносимый формат"}

:::

## Кому подходит Docara

:::card
### Автору документации

Создавайте страницы в Markdown, наследуйте настройки разделов и получайте
готовый статический сайт без базы данных и панели управления.

:::

:::card
### Разработчику

Используйте проверяемые схемы, точный Framework lock и объяснимый файл
`resolved-page-plans.json` для диагностики результата.

:::

## Граница текущего контура

Эта документация описывает проверяемый локальный контур. Она не заявляет
production readiness, готовность всех компонентов Simai Framework или полностью
автономную offline-сборку.
