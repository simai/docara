# Локальный просмотр

Команда `serve` сначала собирает выбранное окружение, а затем запускает
встроенный PHP-сервер.

```bash
php vendor/bin/docara serve local --host=localhost --port=8000
```

Если результат уже собран, добавьте `--no-build`:

```bash
php vendor/bin/docara serve local --host=localhost --port=8000 --no-build
```

Для ServBay или другого локального веб-сервера укажите document root на
содержимое `build_local` или `build_production`. Не направляйте сервер на
Markdown и JSON исходники.

После запуска проверьте корневую страницу, вложенный маршрут, светлую и тёмную
темы, Network и Console.

Для страницы с несколькими H2/H3 дополнительно проверьте:

- desktop: оглавление закреплено справа, а breadcrumbs и previous/next ведут на
  ожидаемые страницы;
- viewport до 1152 px: оглавление доступно через native details над статьёй;
- viewport до 600 px: previous/next складываются в колонку без горизонтального
  overflow;
- Tab показывает заметный focus, Enter открывает ссылки, а переход по
  `#fragment` ставит нужный заголовок в видимую область;
- светлая и тёмная темы сохраняют контраст всех трёх навигационных поверхностей.
