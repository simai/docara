# PHP-only сборка

Для переносимого сайта достаточно PHP 8.2+, Composer и установленного пакета
`simai/docara`.

```bash
php vendor/bin/docara build local
php vendor/bin/docara build production
```

Первый аргумент — имя окружения. По умолчанию используется `local`, а результат
пишется в прямой дочерний каталог `build_<environment>` внутри корня сайта.

Portable builder разрешает очистку только безопасного каталога `build` или
`build_*`. Символическая ссылка, выход из корня или пересечение с входными
файлами останавливают сборку до удаления существующего результата.

Node.js, `.env`, `config.php` и `source/_core` для этого пути не требуются.
