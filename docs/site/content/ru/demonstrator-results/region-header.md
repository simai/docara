# Брендированный header

Область `header` наследует данные бренда сайта. Первая секция выводит составной
Smart-компонент Docara, вторая — кнопку Simai Framework с проверяемыми props.

## Что можно менять

Название, подпись и логотип задаются в `docara.json`, а состав секций и действие
— в JSON страницы или раздела.
