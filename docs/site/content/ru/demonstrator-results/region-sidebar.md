# Sidebar с дополнительным блоком

Sidebar состоит из многоуровневого меню и небольшого текстового блока. Оба
блока остаются частью одной декларативной области и попадают в мобильное меню.

## Адаптивность

На узком экране основной sidebar скрывается, а тот же безопасно клонированный
контент открывается кнопкой меню.
