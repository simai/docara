# Документация и лендинги из Markdown и JSON

Docara собирает статический сайт одной PHP-командой. Содержание остаётся в
Markdown, настройки наследуются из строгих JSON-файлов, а интерфейс строится
на Simai Framework.

:::cta
[Начать работу](/guides/getting-started/)
:::

## Всё необходимое без лишнего runtime

:::features
- **Пишите в Markdown.** Страницы, код, таблицы и типизированные компоненты хранятся рядом с проектом.
- **Настраивайте через JSON.** Бренд, макет и разделы наследуются предсказуемо и проверяются схемой.
- **Собирайте на PHP.** Статический результат детерминирован, а версии Simai Framework зафиксированы.
:::

## От пустой папки до сайта

После установки актуального пакета две команды создают переносимый проект и
собирают локальный результат. До публичного portable release точный способ
установки берите из документации той revision Docara, которую проверяете.

```shell
php vendor/bin/docara init
php vendor/bin/docara build local
```
